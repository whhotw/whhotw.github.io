兩個關鍵字 `import` `export`，用於載入與輸出模組。但實際上不一定會使用到模組的程式碼，這時候就需要動態載入，在確定需要使用的時候載入

```
export let go = () => {
	return 'AAA';
};

async () => {
	let Module = await import('./Module.js');
	console.log(Module.go());
}
```

輸出

```
AAA
```
