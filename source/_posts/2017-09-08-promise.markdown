最早是在社群提出，但 `ES6` 推出後就被列入標準了，原生提供 `Promise` 對象，讓前端工程師處理非同步問題。例如：下載網路上的圖片之後顯示。
#基本用法
```
var promise = new Promise(function(resolve, reject) {
  // ... some code

  if (true){
    resolve(value);
  } else {
    reject(error);
  }
});
```
